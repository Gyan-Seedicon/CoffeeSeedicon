# Seedicon Coffee Exports — Design System

## 1. Design Direction

Seedicon Coffee Exports should feel like a premium, trustworthy, B2B export brand.  
The visual system is inspired by the warm, confident Starbucks-style palette, but adjusted for Indian coffee sourcing, export reliability, quality assurance, and global buyer trust.

The website should feel:

- Premium
- Natural
- Export-ready
- Trustworthy
- Clean and evergreen
- Easy to scan
- Professional, not café-like

Avoid making the website look like a coffee shop, café menu, or lifestyle brand. This is a B2B export landing page.

---

## 2. Color Palette

### Primary Greens

Use green as the main brand language.

| Token | Hex | Use |
|---|---:|---|
| Deep Forest | `#1E3932` | Dark sections, footer, premium feature bands |
| Export Green | `#006241` | Main brand color, headings, important highlights |
| Action Green | `#00754A` | Primary CTA buttons, links, active states |
| Soft Green | `#D4E9E2` | Light badges, subtle cards, form states |

### Warm Neutrals

Use warm backgrounds instead of cold white.

| Token | Hex | Use |
|---|---:|---|
| Warm Cream | `#F2F0EB` | Main page background |
| Ceramic Beige | `#EDEBE9` | Alternate section background |
| White | `#FFFFFF` | Cards, forms, clean content areas |

### Accent

Use gold very carefully as a quality/export accent.

| Token | Hex | Use |
|---|---:|---|
| Quality Gold | `#CBA258` | Certification badges, premium highlights, small accents |
| Gold Wash | `#FAF6EE` | Light background for certification or trust blocks |

### Text

| Token | Value | Use |
|---|---|---|
| Text Strong | `rgba(0,0,0,0.87)` | Main headings and body on light backgrounds |
| Text Muted | `rgba(0,0,0,0.58)` | Supporting copy, captions, metadata |
| Text White | `#FFFFFF` | Text on dark green sections |
| Text White Soft | `rgba(255,255,255,0.72)` | Supporting copy on dark sections |

---

## 3. Typography
 
### Fonts
 
Use Google Fonts.
 
| Role | Font | Reason |
|---|---|---|
| Hero/Display Headings | `Instrument Serif` | Premium, editorial, calm, and refined serif |
| Body & UI Text | `Manrope` | Clean, readable, modern sans-serif |
 
### Type Style
 
Headings should feel premium, editorial, calm, and refined.  
Body text should feel clear, simple, and professional.
 
Suggested scale:
 
| Role | Size | Weight | Line Height |
|---|---:|---:|---:|
| Hero Heading | 56–72px | 400 | 0.9–1.0 |
| Section Heading | 36–48px | 400 (Serif) or 600 (Manrope) | 1.15 |
| Card Heading | 20–24px | 600 | 1.3 |
| Body Large | 18–20px | 400/500 | 1.6 |
| Body | 16px | 400 | 1.6 |
| Small | 14px | 400/500 | 1.5 |
 
Use slightly tight letter spacing for headings: `-0.04em` for Instrument Serif display.

---

## 4. Layout Style

Use a clean one-page landing layout.

### Page Width

- Max container width: `1200px`
- Desktop side padding: `40px`
- Tablet side padding: `24px`
- Mobile side padding: `16px`

### Section Spacing

- Desktop section padding: `96px 0`
- Tablet section padding: `72px 0`
- Mobile section padding: `56px 0`

### Layout Rhythm

Use a clear page flow:

1. Home
2. About Us
3. Coffee Products
4. Sourcing & Origin
5. Quality & Certifications
6. Export Process
7. Contact / RFQ

Keep each section focused and easy to scan.

---

## 5. Components

### Buttons

All buttons should be full-pill.

#### Primary Button

- Background: `#00754A`
- Text: `#FFFFFF`
- Border radius: `999px`
- Padding: `12px 22px`
- Font: `Manrope`, 15–16px, 600
- Hover: slightly darker green or soft lift
- Active: `scale(0.97)`

#### Secondary Button

- Background: transparent
- Text: `#00754A`
- Border: `1px solid #00754A`
- Border radius: `999px`
- Padding: `12px 22px`

#### Button on Dark Section

- Primary: white button with green text
- Secondary: transparent button with white border

---

### Cards

Use cards for products, origin regions, quality checks, and process steps.

Card style:

- Background: `#FFFFFF`
- Border radius: `16px`
- Border: `1px solid rgba(0,0,0,0.06)`
- Shadow: `0 1px 2px rgba(0,0,0,0.06), 0 8px 24px rgba(0,0,0,0.06)`
- Padding: `24px`
- Hover: slight lift and softer shadow

Cards should feel premium but not heavy.

---

### Badges / Chips

Use chips for certifications, coffee types, regions, and export markets.

- Background: `#D4E9E2`
- Text: `#006241`
- Border radius: `999px`
- Padding: `6px 12px`
- Font size: `13–14px`
- Font weight: `600`

Gold chips can be used for certifications only.

---

### Forms

RFQ form should feel simple and professional.

Input style:

- Background: `#FFFFFF`
- Border: `1px solid rgba(0,0,0,0.14)`
- Border radius: `12px`
- Padding: `14px 16px`
- Font: `Manrope`, 15–16px
- Focus border: `#00754A`
- Focus ring: soft green glow

Use clear labels above inputs. Avoid complex floating labels.

---

## 6. Section Design Guidance

### 1. Home

Use a strong hero layout.

- Left: headline, short B2B copy, CTA buttons
- Right: premium plantation / coffee cherries / coffee estate image
- Background: warm cream or deep green split layout

Headline:

“Premium Indian Coffee Exporter Connecting Global Markets”

Hero should immediately communicate trust, sourcing, and export capability.

---

### 2. About Us

Use a 2-column section.

Show:

- Vision in global coffee trade
- Ethical sourcing
- Farmer and estate network
- Reliable supply capability

Use a farm, estate, or farmer partnership image.

---

### 3. Coffee Products

Use a product card grid.

Products:

- Arabica Coffee
- Robusta Coffee
- Specialty Coffee
- Green Coffee Beans
- Roasted Coffee

Each card should have:

- Product name
- Short export-focused description
- Small image or icon
- Optional “Available for bulk supply” note

---

### 4. Sourcing & Origin

Use a map-inspired layout or region cards.

Regions:

- Coorg
- Chikmagalur
- Wayanad
- Araku Valley

Use coffee estate photos and origin-style visuals.

---

### 5. Quality & Certifications

This section should build buyer confidence.

Show:

- Bean grading
- Moisture testing
- Sorting process
- Packaging standards
- Quality inspection

Certification chips:

- FSSAI
- APEDA
- Organic
- Fair Trade
- Rainforest Alliance

Use Quality Gold only for certification highlights.

---

### 6. Export Process

Use a clean 6-step timeline.

Steps:

1. Buyer Inquiry
2. Sample Approval
3. Contract Finalization
4. Production & Quality Inspection
5. Export Documentation
6. International Shipping

Use icons or numbered cards. Keep it simple.

---

### 7. Contact / RFQ

Use a strong final CTA section.

Include:

- Company Name
- Country
- Contact Person
- Email
- WhatsApp
- Coffee Type
- Quantity Required
- Monthly Requirement
- Destination Port

Primary CTA:

“Submit RFQ”

The section should feel serious, trustworthy, and conversion-focused.

---

## 7. Image Direction

Use realistic, premium visuals.

Recommended image types:

- Indian coffee plantations
- Coffee cherries
- Farmers harvesting
- Green coffee beans
- Roasted coffee beans
- Coffee sacks in warehouse
- Quality inspection
- Export container / port logistics

Avoid:

- Café interiors
- Latte art
- Barista lifestyle images
- Overly decorative coffee cup photos
- Generic AI-looking stock images

---

## 8. Micro-Interactions

Use subtle interactions only.

Recommended:

- Smooth scroll to sections
- Button hover lift
- Button active scale `0.97`
- Card hover lift
- Image slight zoom on hover
- Smooth section reveal on scroll
- Clean input focus state

Avoid:

- Heavy motion
- Flashy animations
- 3D effects
- Long loading animations
- Distracting scroll effects

---

## 9. Implementation Notes

Use this design system with:

- React
- Next.js
- TypeScript
- Tailwind CSS
- App Router

Keep components reusable:

- Button
- SectionHeader
- Card
- Badge
- ProductCard
- ProcessStep
- RFQForm

Use Tailwind tokens or CSS variables for all colors, spacing, radius, and shadows.

---

## 10. Design Principle

The final website should make a global buyer feel:

“Seedicon looks reliable, professional, and capable of supplying quality Indian coffee for export.”

Every design decision should support trust, sourcing confidence, quality assurance, and RFQ conversion.
