# Seedicon Coffee Exports — Project Context

## Project Type

This is a **static one-page B2B website** for Seedicon Coffee Exports.

The website should be built using:

* React
* Next.js
* TypeScript
* Tailwind CSS

Use the provided `designsystem.md` file strictly for colors, typography, spacing, buttons, cards, and overall visual style.

## Website Goal

The goal is to build trust with international coffee buyers and encourage them to request samples, pricing, or submit an RFQ.

The website should feel:

* Premium
* Trustworthy
* Export-focused
* Clean
* Evergreen
* Easy to scan
* Simple and professional

Avoid making it look like a café or lifestyle coffee brand. This is a **B2B coffee export website**.

## Main Sections

### 1. Home

Headline:

**Premium Indian Coffee Exporter Connecting Global Markets**

Show clear CTAs:

* Request Sample
* Get Export Pricing
* Contact / RFQ

Use strong visuals like coffee plantations, coffee cherries, or premium coffee estate images.

### 2. About Us

Highlight:

* Seedicon’s vision in global coffee trade
* Ethical sourcing
* Farmer and estate networks
* Reliable sourcing capability

### 3. Coffee Products

Show product cards for:

* Arabica Coffee
* Robusta Coffee
* Specialty Coffee
* Green Coffee Beans
* Roasted Coffee

Keep the copy short, clear, and export-focused.

### 4. Sourcing & Origin

Show Indian coffee-growing regions:

* Coorg
* Chikmagalur
* Wayanad
* Araku Valley

Use origin-style visuals, farm images, and map-inspired layout if needed.

### 5. Quality & Certifications

Show:

* Bean grading
* Moisture testing
* Sorting process
* Packaging standards
* Quality inspection

Mention certifications when available:

* FSSAI
* APEDA
* Organic
* Fair Trade
* Rainforest Alliance

### 6. Export Process

Show a simple 6-step process:

1. Buyer Inquiry
2. Sample Approval
3. Contract Finalization
4. Production & Quality Inspection
5. Export Documentation
6. International Shipping

Use a clean timeline or step-based layout.

### 7. Contact / RFQ

Create a professional RFQ form with:

* Contact Person
* Email
* Phone Number
* Country
* Coffee Type
* Quantity Required
* Export to (destination port)

Primary CTA:

**Submit RFQ**

## Interaction Direction

Use subtle micro-interactions only:

* Smooth scroll
* Soft hover states
* Gentle card elevation
* Subtle image movement
* Clean form focus states

Avoid heavy animations or flashy effects.

## Final Requirement

The page should clearly communicate that Seedicon can source, quality-check, and export Indian coffee reliably for global buyers.
